== Description ==

SKT Gardening Lite is a green theme with focus on nature, eco friendly, herbal, organic, natural, fresh, farm produce, solar energy, landscaping, conservation, environment, agriculture, renewable, recycling, plantation, garden, flowers and trees type of websites like nursery. Ecology, forest, water bodies, earth, bio produce, animal husbandry. Non profit organizations NGOs, charity, nature oriented businesses, wildlife, soulful resorts, far away tours and travels, nature photographers can also use it. Multilingual plugins friendly, translation ready and WooCommerce friendly for eCommerce. Contact forms plugin friendly for call to action. SEO friendly. It has a ready to import 150+ Elementor templates for additional designs. Demo: https://sktthemesdemo.net/gardening/
